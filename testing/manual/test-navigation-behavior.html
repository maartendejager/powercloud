PowerCloud Navigation Behavior Test

This page demonstrates the different navigation behaviors implemented for PowerCloud extension buttons.

Spend Cloud Features (Internal Navigation)

These buttons navigate within the same tab:

  • View Card Details - from entry pages to card pages
  • View Card Book - from card pages to book pages
Implementation: window.location.href = url
Behavior: Same tab navigation (internal Spend Cloud pages)
Button Color: Spend Cloud Blue (#007aca)
Adyen Features (External Navigation)

These buttons open in a new tab:

  • View Card in Adyen - opens Adyen payment instrument page
  • View Balance Account in Adyen - opens Adyen balance account page
  • View Entries in Adyen - opens Adyen transactions page
Implementation: chrome.runtime.sendMessage({ action: "openTab", url: ... })
Behavior: New tab navigation (external Adyen platform)
Button Color: Adyen Green (#0abf53)

Navigation Rationale

The different navigation behaviors are designed to provide the best user experience:

  • Same Tab (Spend Cloud): Users stay within their Spend Cloud workflow when navigating between related internal pages
  • New Tab (Adyen): Users can view external Adyen data while keeping their Spend Cloud session active

Testing the Implementation

To test the navigation behavior:

  1. Load the PowerCloud extension in your browser
  2. Navigate to relevant Spend Cloud pages (entry, card, or book pages)
  3. Click the PowerCloud buttons and observe the navigation behavior
  4. Verify that:
    • Spend Cloud buttons (blue) navigate in the same tab
    • Adyen buttons (green) open in a new tab
